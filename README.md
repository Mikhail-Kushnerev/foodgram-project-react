## Описание

 Онлайн-сервис, где пользователи могут:

- публиковать рецепты;
- подписываться на публикации других пользователей;
- добавлять понравившиеся рецепты в список «Избранное»;
- перед походом в магазин скачивать сводный список продуктов.
